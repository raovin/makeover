# Windhawk taskbar skin

Windhawk is optional. Windows Explorer remains the shell and the normal taskbar
remains responsible for Start, Search, app switching, previews, the system tray,
notifications, Show Desktop, and reserving the desktop work area.

Recommended configuration:

1. Install **Windows 11 Taskbar Styler** by m417z from Windhawk's Explore page.
2. Open the installed mod's **Settings** tab.
3. Select **DockLike** and save.

Do not install a task-switcher replacement, taskbar replacement, dock animation,
window mover, or taskbar-height mod as part of the baseline profile. Each extra
Explorer injection increases the regression surface. DockLike is integrated into
Taskbar Styler and is reversible by disabling the single mod.

Official references:

- https://windhawk.net/mods/windows-11-taskbar-styler
- https://github.com/ramensoftware/windows-11-taskbar-styling-guide/tree/main/Themes/DockLike
